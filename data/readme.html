This directory hosts a variety of data files and subdirectories related to the VersionSpace-in-MeTTa project

Combined Files

  • whole_flybase.datalog.metta(.gz) - (4.1G/382mb): The Combination of all Flybase Precomputed Files in MeTTa
  • whole_flybase.datalog(.gz) - (4.4G/392mb): Datalog file for the above.
  • whole_flybase.qlf(.gz) - (2.0G/364mb): Quick Load Format - Loads all of the above in under 30 seconds!
                                          (In both the offical rust version and the MeTTaLog version)

Contents

  • Individual MeTTa Files Contains the individualized breakdowns of the above files from Flybase precomputed release 2023_5
  • Supplemental - Extra context or information used in our dataset.
    • 12_ontologies - A collection of 12 different ontologies. (contains the individualized breakdowns of the above files)
    • public.pub.tsv - Tab-separated values file of public publications.
  • Extra-Supplemental - Additional supplemental files optional for us to load on demand.
  • TSV Exports - Entire Chado SQL Database in TSV format.
  • Mozi AI - AI models and datasets from mozi.ai.
  • Archive - Historical data and previous versions.
